Covid-19 Rates

Data visualizations regarding Covid-19 deaths in the beginning of the pandemic

Spring 2020

Project Type: Website

Role: Programmer, Designer

Tools: HTML, CSS, R

Overview

This project mainly showcased our team's use of R to create visualizations. We used a CSV file and cleaned and filtered it down to the data we needed to create these interactive visualizations. Each visualization reveals a fact about COVID-19 during the spring of 2020 and users can adjust the filters to drill down onto information they want to view. In addition, we also used HTML and CSS to give our site an interface that will allow users to easily navigate between each visualizations.
